# Команды

У GitPluck есть два уровня команд:

- стартовые флаги, которые передаются в `GitPluck`
- команды интерактивного браузера после загрузки дерева репозитория

## Стартовые флаги

```bash
GitPluck [URL] [--out PATH] [--cwd] [--no-folder] [--token TOKEN|auto|gh]
```

При запуске через SwiftPM:

```bash
swift run GitPluck -- [URL] [--out PATH] [--cwd] [--no-folder] [--token TOKEN|auto|gh]
```

SwiftPM также принимает привычный вариант без `--`:

```bash
swift run GitPluck https://github.com/owner/repo
```

### Аргументы

| Аргумент | Описание |
| --- | --- |
| `URL` | URL GitHub-репозитория. Если не передать, GitPluck спросит его интерактивно. |

Поддерживаемые формы URL:

```text
https://github.com/owner/repo
https://github.com/owner/repo/tree/main/path/to/folder
https://github.com/owner/repo/blob/main/path/to/file
owner/repo
```

### Флаги

| Флаг | Описание |
| --- | --- |
| `--out PATH` | Базовая директория для скачивания. |
| `--cwd` | Использовать текущую директорию как базовую директорию скачивания. |
| `--no-folder` | Скачать прямо в базовую директорию, без подпапки с именем репозитория. |
| `--token TOKEN` | Использовать GitHub token для текущего запуска. |
| `--token auto` | Прочитать token через `gh auth token`. |
| `--token gh` | То же самое, что `--token auto`. |
| `-h`, `--help` | Показать справку. |

## Команды интерактивного браузера

| Команда | Описание |
| --- | --- |
| `<n>` | Выбрать или снять выбор с видимого пункта `n`. |
| `t <n...>` | Выбрать или снять выбор с нескольких видимых пунктов. Пример: `t 1 2 5`. |
| `e <n>` | Зайти в пункт `n`, если это папка; иначе выбрать или снять выбор. |
| `cd <n>` | Зайти в папку `n`. Если `n` — файл, GitPluck покажет ошибку. |
| `l` | Выйти на уровень выше. В корне репозитория ничего не делает. |
| `back`, `b`, `..` | То же самое, что `l`. |
| `/ <text>` | Искать видимые элементы по полному пути в репозитории. |
| `clear` | Очистить активный поиск. |
| `a` | Выбрать все видимые элементы. |
| `u` | Снять выбор со всех видимых элементов. |
| `clear-selection` | Полностью очистить выбор. |
| `selected` | Показать выбранные пути. |
| `p <n>` | Показать preview первых 16 KB текстового файла `n`. |
| `d` | Скачать выбранные элементы. |
| `q`, `quit`, `exit` | Выйти без скачивания. |
| `?`, `help` | Показать справку браузера. |

## Структура скачивания

По умолчанию:

```text
~/Downloads/<repo-name>/<selected-paths>
```

С `--out ./tmp`:

```text
./tmp/<repo-name>/<selected-paths>
```

С `--out ./tmp --no-folder`:

```text
./tmp/<selected-paths>
```
