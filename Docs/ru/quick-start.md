# Быстрый старт

Это руководство показывает самый короткий путь от GitHub URL до скачанного файла.

## Открыть репозиторий

Из корня пакета:

```bash
swift run GitPluck https://github.com/octocat/Hello-World
```

GitPluck выполнит следующие шаги:

1. Разберет GitHub URL.
2. Определит default branch, если ветка не указана в URL.
3. Загрузит дерево репозитория через GitHub API.
4. Покажет интерактивный консольный браузер.

## Выбрать и скачать

В браузере вводите команды в строку `Command:`.

Для репозитория `octocat/Hello-World` виден один файл:

```text
1
d
```

Это значит:

- `1` выбирает пункт 1.
- `d` скачивает выбранные элементы.

По умолчанию GitPluck скачивает в пользовательскую папку Downloads и создает подпапку с именем репозитория.

## Ходить по папкам

Используйте:

```text
e 1
```

Если пункт 1 — папка, GitPluck зайдет в нее.
Если пункт 1 — файл, GitPluck выберет или снимет выбор с него.

Чтобы выйти на уровень выше:

```text
l
```

В корне репозитория `l` ничего не делает.

## Искать по путям

```text
/ Sources
```

Команда фильтрует видимые элементы по полному пути в репозитории.

Очистить поиск:

```text
clear
```

## Выбрать папку скачивания

```bash
swift run GitPluck https://github.com/owner/repo --out ./Downloaded
```

Не создавать подпапку с именем репозитория:

```bash
swift run GitPluck https://github.com/owner/repo --out ./Downloaded --no-folder
```
